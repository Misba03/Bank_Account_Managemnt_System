# Bank Account Management System

A Java-based banking application built with Core Java, JDBC, and MySQL.

## Features
- Create and manage customer accounts
- Deposit, withdraw, and transfer funds
- View complete transaction history
- Secure database connectivity using JDBC

## Technologies Used
- Core Java
- JDBC
- MySQL

## How to Run
1. Clone this repository
2. Set up MySQL and create the database
3. Update DB credentials in the connection file
4. Open in Eclipse or VS Code
5. Run Main.java
